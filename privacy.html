1. 회원가입 및 데이터 수집 개요

📱 회원가입은 선택사항입니다

본 앱은 회원가입 없이도 모든 기본 기능을 이용할 수 있습니다. 회원가입 여부에 따라 수집되는 데이터가 다릅니다.

🔓 비회원 사용자

  • 익명 ID: 앱 설치 시 자동으로 생성되어 기기에 저장됩니다
  • 나눔글: 익명 ID와 함께 서버에 저장됩니다
  • 제한사항: 기기를 변경하거나 앱을 재설치하면 이전 나눔글을 복구할 수 없습니다
  • 개인정보 수집: 이메일, 이름 등 개인을 식별할 수 있는 정보는 수집하지 않습니다

👤 회원 사용자 (선택)

  • 소셜 로그인: 구글, 애플, 카카오톡 계정으로 간편 로그인
  • 나눔글 동기화: 모든 기기에서 나눔글을 동기화하여 볼 수 있습니다
  • 데이터 복구: 기기 변경 시에도 나눔글을 복구할 수 있습니다
  • 추가 수집 정보: 이메일 주소, Firebase UID

2. 개인정보의 처리 목적

"오늘의큐티" (이하 "본 앱")은 다음의 목적을 위하여 개인정보를 처리합니다. 처리하고 있는 개인정보는 다음의 목적 이외의 용도로는 이용되지 않으며, 이용 목적이 변경되는 경우에는 관련 법령에 따라 별도의 동의를 받는 등 필요한 조치를 이행할 예정입니다.

  • 서비스 제공 및 운영 (큐티 콘텐츠 제공, 알림 서비스)
  • 사용자가 작성한 나눔글 및 묵상 내용 저장 및 관리
  • 회원 가입 및 관리 (선택사항 - Firebase 기반 소셜 로그인)
  • 서비스 개선 및 통계 분석

3. 수집하는 개인정보 항목

본 앱은 회원가입 여부에 따라 다음과 같은 정보를 수집합니다:

🔓 모든 사용자 (비회원 포함)

📱 앱 서버에 저장되는 정보

  • 익명 ID (UUID) - 앱 설치 시 자동 생성되어 기기에 저장되며, 나눔글과 연결하기 위해 서버에도 저장됩니다
  • 나눔글 및 묵상 내용 - 사용자가 직접 작성한 텍스트 콘텐츠 (익명 ID 또는 Firebase UID와 연결)
  • 작성 일시 - 나눔글 관리 및 표시 목적

🔍 자동으로 수집되는 정보

  • 기기 정보 (기기 모델, OS 버전)
  • 앱 사용 기록 및 분석 데이터 (Firebase Analytics)
  • 광고 식별자 (Google AdMob 광고 제공을 위해)

👤 회원 사용자만 추가 수집 (선택사항)

🔐 소셜 로그인 정보 (Firebase에서 관리)

  • 이메일 주소 - 소셜 로그인 제공자(구글, 애플, 카카오톡)로부터 수신하여 Firebase Authentication에서 관리
  • 소셜 로그인 제공자 정보 - 어떤 서비스(구글/애플/카카오톡)로 로그인했는지에 대한 정보
  • 프로필 정보 - 소셜 로그인 제공자가 제공하는 기본 프로필 정보 (이름 등, 선택적)
  • Firebase UID - Firebase가 생성한 고유 사용자 식별자 (앱 서버에 저장)

중요: 본 앱은 비밀번호를 수집하거나 저장하지 않습니다. 구글, 애플, 카카오톡 계정으로 간편하게 로그인하며, 모든 인증은 각 소셜 로그인 제공자와 Firebase Authentication을 통해 안전하게 처리됩니다. 이메일 주소와 인증 정보는 본 앱 서버에 저장되지 않습니다.

4. 개인정보의 처리 및 보유기간

본 앱은 법령에 따른 개인정보 보유·이용기간 또는 정보주체로부터 개인정보를 수집 시에 동의받은 개인정보 보유·이용기간 내에서 개인정보를 처리·보유합니다.

비회원 사용자

  • 익명 ID: 앱 삭제 시까지 (앱 삭제 시 서버에서도 삭제 가능하도록 기능 제공)
  • 나눔글: 사용자가 직접 삭제하거나 앱 삭제 요청 시까지
  • 앱 사용 기록: 수집일로부터 3개월
  • 광고 관련 데이터: Google AdMob 정책에 따름

회원 사용자

  • Firebase UID: 회원 탈퇴 시까지 (탈퇴 즉시 앱 서버에서 삭제)
  • 나눔글 및 묵상 내용: 회원 탈퇴 시까지 (탈퇴 시 사용자 선택에 따라 삭제 또는 익명 처리)
  • 이메일 주소 (Firebase): 회원 탈퇴 시까지 (Firebase에서 관리)
  • 앱 사용 기록: 수집일로부터 3개월
  • 광고 관련 데이터: Google AdMob 정책에 따름

5. 개인정보의 제3자 제공

본 앱은 원칙적으로 사용자의 개인정보를 외부에 제공하지 않습니다. 다만, 아래의 경우에는 예외로 합니다:

  • 사용자가 사전에 동의한 경우
  • 법령의 규정에 의거하거나, 수사 목적으로 법령에 정해진 절차와 방법에 따라 수사기관의 요구가 있는 경우

📍 데이터 저장 위치

데이터 유형 비회원 회원 저장 위치

🔍 익명 ID란?

익명 ID는 사용자를 식별하기 위한 무작위로 생성된 고유 식별자입니다. 이메일이나 이름 같은 개인정보와는 달리 개인을 특정할 수 없으며, 오직 나눔글을 저장하고 불러오는 용도로만 사용됩니다.

  • 앱 최초 실행 시 자동 생성
  • 기기에 안전하게 저장
  • 개인을 식별할 수 없는 무작위 문자열 (예: "a1b2c3d4-e5f6-7890-abcd-ef1234567890")
  • 회원가입 시 Firebase UID로 대체

8. Google AdMob 광고 서비스

본 앱은 Google AdMob을 통해 광고를 제공하며, 이 과정에서 다음과 같은 정보가 처리될 수 있습니다:

수집되는 정보

  • 광고 식별자 (Advertising ID)
  • 기기 정보 및 IP 주소

  • AdMob 및 광고: https://support.google.com/admob/answer/6128543

9. 정보주체의 권리·의무 및 행사방법

정보주체는 개인정보주체로서 다음과 같은 권리를 행사할 수 있습니다:

  1. 개인정보 열람 요구
  2. 개인정보 오류 등이 있을 경우 정정 요구
  3. 개인정보 삭제 요구
  4. 개인정보 처리정지 요구

권리 행사는 앱 내 '설정 > 계정 관리' 메뉴를 통해 직접 하실 수 있으며, 이메일 등을 통하여 하실 수 있고 본 앱은 이에 대해 지체없이 조치하겠습니다.

비회원 사용자의 데이터 삭제

비회원 사용자는 앱 내에서 작성한 나눔글을 직접 삭제할 수 있으며, 모든 데이터를 삭제하려면 개인정보 보호책임자에게 익명 ID를 제공하여 요청할 수 있습니다.

10. 개인정보의 파기

본 앱은 개인정보 보유기간의 경과, 처리목적 달성 등 개인정보가 불필요하게 되었을 때에는 지체없이 해당 개인정보를 파기합니다.

파기절차 및 방법

비회원 사용자

  • 데이터 삭제 요청 시:
    • 익명 ID 및 연결된 나눔글 데이터가 즉시 앱 서버에서 삭제됩니다
    • 삭제 후 복구가 불가능합니다

회원 사용자

  • 회원 탈퇴 시:
    • Firebase UID 및 나눔글 데이터는 즉시 앱 서버에서 삭제됩니다
    • 이메일 주소 및 소셜 로그인 연동 정보는 Firebase에서 삭제됩니다
    • 구글, 애플, 카카오톡 계정 자체는 각 서비스 제공자가 관리하므로 삭제되지 않습니다
  • 데이터 파기: 복구 불가능하게 영구 삭제됩니다
  • 로그 데이터: 3개월 후 자동으로 삭제됩니다
